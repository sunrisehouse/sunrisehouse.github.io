1. Linear Equations in Linear Algebra

linear eqaution

coefficient: 계수

consistent/inconsistent: 해 있다 / 없다

coefficient matrix: 계수로만 이루어진 행렬

augmented matrix: 결과값도 붙은 행렬

row operation

  • replacement: 다른 항 곱해서 더한거
  • interchange: 행 위치 변경
  • scaling: 상수곱 (non zero)

row equivalent: row operation 으로 만들 수 있으면

augmented matrix 에서 row equivalence 하면 같은 해 갖는다.

square matrix: nxn matrix

diagonal matrix: square matrix and 대각선 빼고 다 0

  • main diagonal

trace

transposed matrix: 행 열 위치 변경

symmetric matrix: tranpose 해도 같을 때

triangular matrix: main diagnal 기준으로

  • upper: 위에 다 0
  • lower: 아래 다 0

echelon form

  1. 0이 아닌 모든 행은 성분이 모두 0인 행들 위에 항상 놓여 있다.
  2. 각 행의 선행성분들은 그 행보다 위에 있는 행의 선행성분보다 오른쪽에 있는 열에 있다.
  3. 선행성분 밑의 열의 성분들은 모두 영(0)이다.

reduced echelon form

  1. 영(0)이 아닌 행의 선행성분은 1이다.
  2. 각 선행성분 1은 그 열 안에서 영(0)이 아닌 유일한 성분이다.

기약사다리꼴의 유일성 정리: 각 행렬은 오직 하나의 기약 행사다리꼴 행렬과 행동치이다.

pivot position: reduced 에서 leading 하는 1 자리

pivot column: pivot position 의 column

  • basic variable: pivot column 에 해당
  • free variable: 그 외

augment matrix 에서 마지막 컬럼이 pivot column 이면 inconsitent (해 없다) (0 = 3 꼴 이라서)

consistent 인 경우 free variable 없으면 unique (해 있는 경우에 모든 column 다 leading 1 갖고 있으면 해 1개 나머지는 해 무한대)

해 구하는 방법

  1. echelon form 으로 해 유무 조사 (pivot column)
  2. reduced echelon form 으로 해 조사

linear combination: vector 들의 합

span: vector 들의 집합이 있을 때 그 원소들을 이용해 더하고 빼서 만들어지는 공간

Ax: A 의 컬럼들을 백턱라고 생각했을 때 x 들의 entry 를 weight 로 갖는 linear combination 으로 생각 가능

homogeneous: Ax = 0 으로 쓰여있는 시스템 (x = 0 이라는 해를 무조건 갖는다)

  • trivial: x = 0 인 해
  • non trivial: x != 0 인 해
